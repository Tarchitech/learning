{# Jinja2 Syntax: {% block title %} - Defines a block that can be overridden by child templates - Child templates can override this title using {% block title %}...{% endblock %} - Default title is "Flask Template Example" #} {% block title %}Flask Template Example{% endblock %}
{# Jinja2 Syntax: {{ url_for('index') }} - url_for() is a Flask function that generates URL based on route function name - 'index' is the route function name (def index()) - Generates: / (home page URL) #} Home Users Create User Products
{# Jinja2 Syntax: {% block content %} - Defines a content block that child templates can fill - Child templates use {% block content %}...{% endblock %} to fill this block #} {% block content %}{% endblock %}
