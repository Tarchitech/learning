Welcome to Flask Template Example

This is a simplified learning example demonstrating Flask + Jinja2 core concepts.

Core Concepts:

  • Template Inheritance: Use {% raw %}{% extends "base.html" %}{% endraw %} to inherit base template
  • Variable Output: Use {% raw %}{{ variable }}{% endraw %} to output variables
  • Loops: Use {% raw %}{% for item in items %}{% endraw %} to iterate lists
  • Conditions: Use {% raw %}{% if condition %}{% endraw %} for conditional logic
